# Hospital-Management-System

A simple Hospital Management System built using Flask and SQLite.  
The application supports Admin, Doctor, and Patient roles with basic appointment and profile management features.

## Requirements
Install all dependencies using:

pip install -r requirements.txt

## How to Run
Start the application with:

python app.py
The server will run at:
http://127.0.0.1:5000
